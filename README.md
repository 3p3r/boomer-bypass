# boomer-bypass

Boomer Bypass is a proxy to defeat Boomer IT people at their job, which they know nothing about.

## Synopsis

Boomer Bypass is a simple HTTP proxy server that routes requests through your locally installed web browser. This allows you to bypass any restrictions or monitoring that your IT department may have in place, as the traffic will appear to be coming from your browser rather than the proxy server.

The purpose of this project is to provide a simple and effective way for users to access the internet without being hindered by their IT department's lack of knowledge and outdated practices.

## Stack

- [mitm-proxy](https://github.com/joeferner/node-http-mitm-proxy)
- [puppeteer](https://pptr.dev)

## Usage

Download the latest release binary for your operating system from the [releases page](https://github.com/3p3r/boomer-bypass/releases)

- `bb-win`: Windows 64 bit
- `bb-mac`: macOS 64 bit / AArch64
- `bb-linux`: Linux 64 bit / AArch64
